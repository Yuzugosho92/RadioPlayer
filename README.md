<B>ラジオ風プレイヤー</B>

音声合成ソフト「VOICEVOX」を使って、手持ちの音楽ファイルにMCをつけて楽しむプレイヤーです。

※注意点<br>
このソフトはまだ開発途上です、細かい例外処理はおこなっておりません。<br>
予期せぬ強制終了が起こるかもしれませんので、予めご了承ください。

◇使い方
1. VOICEVOXを起動します。
2. RadioPlayer1.exeを起動します。
3. 右側の「ここに音楽ファイルを投入」の枠に、音楽ファイルをドロップします。
   （現在、MP3・MP4・FLACのみ対応）
4. 読み込みが完了すると、音楽とMCが流れます。
5. ウィンドウを閉じると、設定が保存されます。
