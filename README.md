<B>ラジオ風プレイヤー</B>

音声合成ソフト「VOICEVOX」を使って、手持ちの音楽ファイルにMCをつけて楽しむプレイヤーです。

※注意点<br>
このソフトはまだ開発途上です、細かい例外処理はおこなっておりません。<br>
予期せぬ強制終了が起こるかもしれませんので、予めご了承ください。

◇使い方
1. VOICEVOXを起動します。
2. RadioPlayer1.exeを起動します。
3. 右側の「ここに音楽ファイルを投入」の枠に、音楽ファイルをドロップします。
   （現在、MP3・MP4・FLAC・Wave・m4aのみ対応）
4. 読み込みが完了すると、音楽とMCが流れます。
5. ウィンドウを閉じると、設定が保存されます。

◇左下「タイミング調整」解説<br>
StartTime <br>
音楽の再生を始める位置を秒で指定します。 <br>

EndingTime<br>
音楽の再生を終わる位置を秒で指定します。<br> 

IntroTime<br>
冒頭でトークを始めるタイミングを秒で指定します。<br>StartTimeと同じ値にした場合、トークはしません。 <br>なおVOICEVOXが喋るまで数秒のタイムラグがあります。<br>

OutroTime<br>
最後にトークを始めるタイミングを秒で指定します。<br>終了する何秒前に始めるかの値です。<br>

IntroMaxLength:<br>
冒頭トークの最大の長さを文字数で指定します。<br>イントロが短く、長いトークだとはみ出す場合に指定してください。<br> 0の場合、長さの制限はしません。<br>

◇音楽を再生中、エンディングトークを入れたいタイミングで「この時間！」ボタンを押すと、EndingTimeの値が設定されます。

◇ジングル曲の導入
1. ジングルにしたい10秒程度の音楽ファイルを別途用意します。
2. ラジオ風プレイヤーに音楽ファイルをドロップします。
3. 右下のリストに曲が追加されますので、右クリックをして「編集(E)」を選びます。
4. 基本情報の「Type」を「Jingle」に変更して、OKボタンを押します。
